## Vision

Boojy Audio is a free, open-source, cross-platform DAW designed to fill the gap between beginner-friendly but limited tools (GarageBand) and powerful but overwhelming professional software (Ableton, Logic, Pro Tools). 

Part of the larger Boojy creative software suite, it aims to be the first DAW that is simultaneously accessible to beginners and capable enough for serious production.

---

## Core Principles

- **Performance first** â€” Runs smoothly on modest hardware
- **Minimal but complete** â€” Every feature polished, nothing half-done
- **Progressive disclosure** â€” Simple by default, powerful when needed
- **Cross-platform** â€” Same experience on Mac, Windows, Linux, and Web
- **Ecosystem thinking** â€” Designed as part of the Boojy suite from day one

| App | Status Letter | Connected | Offline | Error |

| Boojy Video | O (filled) | Blue | Yellow | Red |
| Boojy Design | D | Blue | Yellow | Red |

## Design References

Each major feature draws inspiration from the best existing implementation:

| Feature | Primary Reference | Reasoning |
|---------|-------------------|-----------|
| Piano Roll | FL Studio | Gold standard â€” ghost notes, scale highlighting, intuitive interactions |
| Arrangement View | Studio One | Draggable sections, scratch pads, excellent drag-and-drop |
| Audio Recording | Logic Pro | Excellent comping, beginner-friendly, professional results |
| Audio Editing/Warping | Ableton Live | Best-in-class warping, intuitive, sounds good |
| Automation | Studio One / Bitwig | Inline lanes below tracks, no mode switching, multiple visible |

| Stock Effects | Ableton Live | Simple interfaces, hard to mess up, good defaults |

## Version 1.0 â€” Core Release

**Target:** December 2025  
**Platforms:** macOS, Windows, Linux, Web

### Arrangement & Editing

- Arrangement view with Studio One-style draggable sections
- Piano roll with FL Studio-inspired features
- Audio recording with multi-track and comping support
- Audio editing with single-mode auto-detecting time stretch
- Inline automation lanes (Studio One/Bitwig approach)
- Track grouping and custom track colors
- Freeze/flatten with non-destructive undo (keeps original hidden)
- Markers and arrangement markers
- Bouncing/rendering in place
- Stem separation
- Tempo and time signature automation

### Built-in Instruments

- **Boojy Synth** â€” Simplified Serum-style wavetable synth
- **Boojy Sampler** â€” Simple/Advanced toggle, one instrument for all sampling needs
- **Boojy Drums** â€” Pad grid + step sequencer hybrid
- **Preset Instrument Player** â€” For bundled sounds (piano, strings, etc.)

### Effects & Processing

- Core audio effects (EQ, compressor, reverb, delay, limiter, etc.)
- Core MIDI effects
- Sidechaining via dropdown in compressor (Logic-style)
- Per-effect bypass button (top-left) + global track FX bypass

### Mixing & Mastering

- Track mixer with sends/returns (2 default, expandable)
- Mastering Assistant with LUFS metering and platform targets
- Reference track comparison with auto-volume matching
- Peak meters on all tracks, spectrum analyzer on master

### Plugin Support

- VST3 support (all platforms)
- AU support (macOS)

### MIDI & Controllers

- External MIDI controller support
- MIDI learn for parameter mapping

### Library & Browser

- Organized library panel with search
- Browser preview with auto-tempo sync
- Genre templates (Lo-Fi, Hip-Hop, Pop, Electronic, etc.)

### Project Management

- Track templates/presets (save and reuse track configurations)
- Configurable undo history (default 100-200, up to unlimited)
- Project snapshots (named versions within a project)
- Autosave and project recovery
- Boojy Cloud sync for projects across devices

### Export & Collaboration

- Export: WAV, MP3, FLAC
- Stem export presets for cross-DAW collaboration
- Clean MIDI import/export

## Target Market

**Primary:**
- Beginners who find existing DAWs overwhelming
- Students who can't afford expensive software
- Hobbyists who want something better than GarageBand but cross-platform
- Content creators who need more than Audacity

**Secondary:**
- Semi-pro producers who want a lightweight alternative
- Linux users (chronically underserved)
- Educators looking for free software for teaching
- Podcasters (with Podcast Mode)

---

## Success Metrics

- Downloads and active users
- Project completion rate (do users finish songs?)
- User retention (do they come back?)
- Community engagement (templates shared, presets created)
- Cross-platform usage (same user on multiple devices)

---

## Risks & Mitigations

| Risk | Mitigation |
|------|------------|
| Scope creep | Strict v1 feature freeze, defer to v2+ |
| Jack of all trades | Nail Produce mode before expanding to Live/Library |
| Performance issues | Rust audio engine, optimize early |
| Plugin compatibility | Extensive VST3/AU testing before launch |
| Discoverability | YouTube tutorials, community building, open source visibility |
